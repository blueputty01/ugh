# 🚗 Uber Global Hackathon

React & SASS /Express configuration using webpack bundler.

## Installation and Running

Starting from root,

### Server

`cd server`

#### Installation

`npm i`

#### Commands

Live server: `npm run dev`
Start server: `npm run start`

### Client

`cd client`

#### Installation

`npm i`

#### Commands

Dev server: `npm run start`
Build client: `npm run build`

## ⚙️ Technologies:

### 💻 Client:

- React
- Tailwind CSS

### 🌐 Server:

- Express.js
